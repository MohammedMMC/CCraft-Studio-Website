# CCraft Studio Website

This repository is a website for the **CCraft Studio** project.

Project source:
https://github.com/MohammedMMC/CCraft-Studio

## Overview

CCraft Studio is a web platform for browsing, sharing, and managing CC: Tweaked projects.

## Pages

- `/` - Home
- `/community` - Public project feed
- `/dashboard` - User dashboard
- `/pp` - Privacy Policy
- `/tos` - Terms of Service
